# vue-cart

> 一个vue版购物车

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 需求

- 显示商品的文字描述、图片描述、类型、价格信息；
- 改变商品颜色的时候图片切换；
- 改变商品类型的时候价格变化；
- 加入/移除购物车；
- 购物车中商品数量统计以及总价的统计；
